# ERP Inventory Management System

Modern ERP Inventory Management System built using:

- Java
- Spring Boot
- Vaadin
- MySQL
- JPA/Hibernate

## Features

- Employee Management
- Inventory Management
- Inventory Requests
- Multi-Level Approval Workflow
- Issue & Return Management
- Dashboard Analytics
- Transaction History
- Role Based Access Control

## Roles

- SUPER_ADMIN
- INVENTORY_ADMIN
- MANAGER
- EMPLOYEE

## Tech Stack

- Spring Boot
- Vaadin
- MySQL
- Maven
- Hibernate
